# connect-four-game
This project is a Python-based implementation of the classic Connect Four game. It is a two-player strategy game where players take turns dropping colored discs into a vertical grid
📌 Description

This project is a Python-based implementation of the classic Connect Four game, where a human player competes against an AI opponent. The game is played on a grid where players take turns dropping discs into columns, aiming to connect four discs in a row.

The AI is designed to make intelligent decisions based on the current state of the board, providing a challenging gameplay experience.

🎯 Objectives
To implement a classic board game using Python
To design an AI opponent for decision-making
To apply problem-solving and logical thinking
To simulate human vs computer interaction

🤖 AI Logic
The AI analyzes the board before making a move
It can:
Block the player’s winning move
Try to create its own winning sequence

Uses the Minimax algorithm (with/without alpha-beta pruning) for optimal decisions

⚙️ Features
Human vs AI gameplay
Smart move selection by AI
Win and draw detection
Input validation
